# 4 docker镜像相关操作

# 4.1 搜索官方仓库镜像

$ docker search centos
NAME                      DESCRIPTION                    STARS    OFFICIAL               AUTOMATED
centos                    The official build of CentOS.  3992     [OK]      
ansible/centos7-ansible   Ansible on Centos7             105
1
2
3
4

# 列表说明

参数 说明
NAME 镜像名称
DESCRIPTION 镜像说明
STARS 是否是官方的
AUTOMATED 是否是自动构建的
